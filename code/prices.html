Packages We Offer

Standard 3 Hour: Free
Our STANDARD package is a 3 hour package where you can book the cinema for three hours. This package takes into account time taken to set up the movie.


Event 6 Hour: $20 per person
Our EVENT package is a 6 hour package where you can book the cinema for six hours. This package takes into account time taken to set up the movie. This package also includes three snack items and one drink from the menu for each person registered for the event. More food can be purchased at the venue during the booking time.


All Day: $50 per person
Our ALL DAY package is a 12 hour package where you can book the cinema for twelve hours. This package takes into account time taken to set up the movie. This package also includes two meals and one snack time from the menu with unlimited drinks. This applies to each person who has been registered for the event. More food can be purchased at the venue during the booking time.
